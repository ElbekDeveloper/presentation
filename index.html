Say no to floats and positioning!

Using CSS Grid Layout, we can develop a layout easily, using rows and columns

So, let's look at some grid properties!

Here is the styling of grid-container:


							.grid-container{
								display: grid; 
 
								grid-template-columns: repeat(3, 1fr);
							}	
						
						

								.grid-container{
									display: grid; 
									grid-template-rows: minmax(100px, auto);
								}			
							

									.grid-container{
										display: grid;
										grid-template-columns: repeat(3, 1fr);
										grid-template-rows: minmax(100px, auto);	
										grid-gap: 5px;				
									}

We can manipulate grid items using grid-line property.

Here is the styling of grid-item:


							
										.grid-item{ 

											grid-row-start: 1;
											grid-row-end: 3;
											//this is equavalent grid-row: 1/ span 2;
										}

Here is the styling of grid-item:


									
										.grid-item{ 
											grid-column-start: 1;
											grid-column-end: 3;	
											//this is equavalent grid-column: 1/ span 2; 
										}
